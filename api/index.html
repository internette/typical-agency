Intro

Welcome to Typical.Agency's API documentation page. This is a RESTful api built by Antoinette Janus that powers the website typical.agency, a project created also by Antoinette Janus.

Why should I use this?

Ever been stuck in with a prototype or wireframe and needed dummy content, content that expands beyond that of "Lorem ipsum dolor sit amet..."? Wonder how all those sponsor logos that your client wants to showcase will look when actually implemented? Need to sell your design, but don't want to spend hours searching for work-appropriate content? Look no further! This API provides Creative Commons images, buzzwords, and over twenty recognizable logos.

Endpoints

The Typical.Agency REST API contains 5 main endpoints: bios, clients, images, colors, and words. Below are the endpoints and the outputs.

/bios

Type: GET

This endpoint provides generic biographies for any company as well as a correlating image. These can be used as slogans, mottos, and simple About blurbs.

Example

GET: http://typical.agency/api/bios

{ "_id":"5732700d09038805987fcc37", "name":"bios", "bios":[{ "tagline":"Collaboration Powered", "bio":"We're a small group that loves what we do and doing it together.", "img":"https://static.pexels.com/photos/7096/people-woman-coffee-meeting.jpg" }] }
